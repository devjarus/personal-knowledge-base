# Contributor Covenant Code of Conduct

This project adopts the **Contributor Covenant Code of Conduct, version 2.1**,
as its official code of conduct.

The full text is canonical and maintained upstream. Please read it here:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Plain-text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md>

By participating in this project — contributing code, opening issues, reviewing
pull requests, or interacting in any project space — you agree to uphold that
Code of Conduct.

## Scope

This Code of Conduct applies within all project spaces (the GitHub repository,
issues, pull requests, discussions) and when an individual is officially
representing the project in public spaces.

## Reporting

If you experience or witness behavior that violates this Code of Conduct,
please **open a GitHub issue in this repository**. If the matter is sensitive
and you prefer not to file a public issue, open a minimal public issue asking
the maintainer to contact you, and we will coordinate a private channel from
there.

All reports will be reviewed and investigated. Project maintainers are
obligated to respect the privacy and security of the reporter of any incident.

## Enforcement

Project maintainers are responsible for clarifying and enforcing the standards
of acceptable behavior. They may take any action they deem appropriate,
including warning, temporary ban, or permanent ban from project spaces, in
response to behavior they deem inappropriate, threatening, offensive, or
harmful.
